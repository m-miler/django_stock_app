Twitter
===========

Twitter functionality allows to get the recent tweets for selected stock company.
To get access to the Twitter API relevant credentials must be provided in .env file.

* Twitter credentials:
    * API_KEY
    * API_KEY_SECRET
    * BEARER_TOKEN
    * ACCESS_TOKEN
    * ACCESS_TOKEN_SECRET


.. note:: Twitter API credentials can be get by creating an account on site https://developer.twitter.com/
